Digital Forensic Triage Tool

A cross-platform GUI application for automated artifact collection on Windows & Linux systems

Project Overview

The Digital Forensic Triage Tool is a Python-based application designed to automate the collection of critical forensic artifacts from Windows and Linux systems. It provides investigators with a quick and efficient way to gather evidence during initial incident response.

The Problem

Manual artifact collection is time-consuming and error-prone. Investigators needed a tool that could:

  • Quickly collect common forensic artifacts
  • Work across multiple operating systems
  • Maintain chain of custody
  • Be easy to use for non-technical investigators

The Solution

I developed a GUI-based triage tool that automates the collection process while maintaining forensic soundness.

Key Features

  • Cross-platform support (Windows & Linux)
  • Automated artifact collection
  • Hash verification for chain of custody
  • Export in multiple formats
  • Modular architecture for easy updates

Technologies Used

Python PyQt5 Plaso pytsk3 SQLite
